NAME
       dcmgpdir - Create a general purpose DICOMDIR


SYNOPSIS
       dcmgpdir [options] [dcmfile-in...]

DESCRIPTION
       The  dcmgpdir  tool is deprecated. Use dcmmkdir instead, which supports
       the same command line parameters, and more.
