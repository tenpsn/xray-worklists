NAME
       cda2dcm - Encapsulate CDA file into DICOM file format


SYNOPSIS
       cda2dcm [options] cdafile-in dcmfile-out

DESCRIPTION
       The  cda2dcm  tool  is deprecated. Use dcmencap instead, which supports
       the same command line parameters, and more.

       dcmencap(1)
