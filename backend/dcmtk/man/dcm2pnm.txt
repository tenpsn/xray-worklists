NAME
       dcm2pnm - Convert DICOM images to PGM/PPM, PNG, TIFF or BMP


SYNOPSIS
       dcm2pnm [options] dcmfile-in [bitmap-out]

DESCRIPTION
       The dcm2pnm tool is deprecated. Use dcm2img instead, which supports the
       same command line parameters, and more.
