NAME
       stl2dcm - Encapsulate STL file into DICOM file format


SYNOPSIS
       stl2dcm [options] stlfile-in dcmfile-out

DESCRIPTION
       The  stl2dcm  tool  is deprecated. Use dcmencap instead, which supports
       the same command line parameters, and more.

       dcmencap(1)
