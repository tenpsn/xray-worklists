NAME
       dcm2cda - Extract CDA file from DICOM encapsulated CDA


SYNOPSIS
       dcm2cda [options] dcmfile-in cdafile-out

DESCRIPTION
       The  dcm2cda  tool  is deprecated. Use dcmdecap instead, which supports
       the same command line parameters, and more.
