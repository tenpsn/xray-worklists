NAME
       pdf2dcm - Encapsulate PDF file into DICOM file format


SYNOPSIS
       pdf2dcm [options] pdffile-in dcmfile-out

DESCRIPTION
       The  pdf2dcm  tool  is deprecated. Use dcmencap instead, which supports
       the same command line parameters, and more.

       dcmencap(1)
