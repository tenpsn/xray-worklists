NAME
       dcm2pdf - Extract PDF file from DICOM encapsulated PDF


SYNOPSIS
       dcm2pdf [options] dcmfile-in pdffile-out

DESCRIPTION
       The  dcm2pdf  tool  is deprecated. Use dcmdecap instead, which supports
       the same command line parameters, and more.
